# 🪷 Сайт Шрилы Хришикеши Махараджа

Современный сайт на **Next.js 14** (App Router) + **Tailwind CSS**.

## Быстрый старт

```bash
# 1. Установить зависимости
npm install

# 2. Запустить в режиме разработки
npm run dev

# 3. Открыть браузер
http://localhost:3000
```

## Деплой на Vercel (бесплатно)

1. Загрузите проект на [GitHub](https://github.com)
2. Зайдите на [vercel.com](https://vercel.com) и нажмите **New Project**
3. Импортируйте репозиторий — Vercel всё определит автоматически
4. Нажмите **Deploy** — сайт будет готов через ~2 минуты

## Что нужно заменить

### Фотографии
Все `img-placeholder` блоки — это заглушки. Чтобы добавить реальные фото:

```tsx
// Вместо заглушки используйте тег Image из next/image:
import Image from 'next/image'
<Image src="/photos/maharaja.jpg" alt="Хришикеша Махарадж" fill className="object-cover" />
```

Положите фото в папку `/public/photos/`.

### Ссылки
Замените в коде:
- `https://t.me/` → ваш Telegram-канал
- `https://youtube.com/` → ваш YouTube-канал
- `https://vk.com/` → ваша группа ВКонтакте

### Тексты
Тексты описаний находятся прямо в компонентах — редактируйте их по своему усмотрению.

## Шрифты

- **Cormorant Garamond** — заголовки, цитаты, санскрит (элегантный серифный)
- **Nunito** — основной текст (мягкий, читаемый)

Загружаются из Google Fonts автоматически.
